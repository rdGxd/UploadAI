## 💻 Projeto

Aplicação que possibilita realizar upload de videos e por meio de IA, criar automaticamente títulos chamativos e descrições com um boa indexação.

---

## 💻 Tecnologias Usadas

## Back-end

Node.js <br/>
TypeScript <br/>
Fastify (Framework) <br/>
Prisma + SQLite <br/>
OpenAI <br />

## Front-end

Vite <br />
React <br/>
Radix <br/>
TailwindCSS <br/>
